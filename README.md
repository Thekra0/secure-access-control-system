# Secure Access Control System

A cybersecurity project that demonstrates secure user authentication and Role-Based Access Control (RBAC) using cryptographic techniques.

## Features

* Secure login system
* Password hashing with PBKDF2-HMAC-SHA256
* Random salt generation
* Role-Based Access Control (RBAC)
* Access logging
* Modern HTML/CSS dashboard
* Python backend demonstration

## Technologies

* Python
* HTML
* CSS
* JavaScript
* PBKDF2-HMAC-SHA256
* Role-Based Access Control (RBAC)
